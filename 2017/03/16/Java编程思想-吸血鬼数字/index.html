Java编程思想-吸血鬼数字

《Java 编程思想》上的一道练习题，虽然看起来简单，但是写一个穷举遍历法都花了好长时间，，真是太渣了。。。

我的穷举遍历法

别人写的利用了Java内置工具类库，效率未必有多高，代码确实简洁了好多

官方标准答案，也不过如此啦

其中利用了一个已知的公理：如果两个数字a，b 是吸血鬼数的尖牙，那么它必然符合：
(a * b) % 9 == (a + b) % 9，但是符合这个条件的却未必就是吸血鬼数的尖牙，所以这个公理只能做一下初期简化！

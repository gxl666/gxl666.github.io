2016 年终总结

一年一度，个人总结。

读书篇

已读完

  1. 《图解 HTTP》：感觉就像是 HTTP 协议的科普书，所以没写任何笔记，具体学习还是《HTTP 权威指南》
  2. 《Spring 实战（第三版）》：解释了很多 Spring 里面基础概念，自己还是写了笔记总结的
  3. 《Head First Java》：八月份入职计划第一波要读的书，刚刚终于看完了，心情真是像书中最后一章的这幅插图：

    坑爹的作者，后面还有两个附录：
  4. 《阿Q正传》：估计是今年唯一读的一本文学类书籍

遗憾的未读完（接下来还是要努力看完目录）

  1. 《深入理解 Java 虚拟机》：简单的地方很简单，复杂的地方就是天书，借来的书看了有两个月了吧，看完应该能提高装逼水平
  2. 《编写高质量代码：改善Java程序的151个建议》：入职计划第二波要读的书，到现在也就看了两章吧。实用性不强，没能吸引我读完
  3. 《HTTP 权威指南》：买了一年了的书，很好很强大，之前读了一遍啥也没记住，最近又跳着读了几章收获颇丰
  4. 《Pro Git》：前面几章简单的基本上都学会了，后面复杂的不懂的还是没学会
  5. 《Netty 权威指南》：两个月前就决定学习 Netty 了，现在还处于打好 IO 操作原理、Socket 编程的基础阶段
  6. 《鸟哥的 Linux 私房菜（基础篇）》：一年前就借别人的书粗略的读过一遍，命令太多了，不常用记不住，接下来还得一部分一部分的读
  7. 《鸟哥的 Linux 私房菜（服务器篇）》：这个就是自己花八十大洋买的了，两个月前读了四五章之后一直在家里吃灰
  8. 《深入剖析 Tomcat》：好多年前的一本书了，无意中发现的，很好很强大啊，我已经读到第六章了
  9. 《Maven 实战》：入职后懵懵懂懂的用了两个月的 Maven，看了这个之后才略懂其原理，蛋疼的是还有几章没读完
  10. 《Android编程权威指南》：工作需要读了几章，不得不说对于安卓开发小白来说真是很好很强大

今年主要读了的书就上面这些吧（貌似更主要的是下半年入职后为了应付工作不得不读的），平时在家也读了点其他方面的书，没啥营养的就不提了吧！

生活篇

2016 过去了，还是单身狗，只能在 2017 里做更多有意义的事情了！

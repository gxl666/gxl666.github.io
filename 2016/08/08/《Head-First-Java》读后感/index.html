《Head First Java》书评

最近重头学习Java，选择了《Head First Java》（中文版）这本书来作为学习资料。这里记录下个人对这本书的看法吧。

我要吐槽

这本书总共十八章，现在看了前八章。感觉这本书很一般，哦不，是很烂啊。那么它烂在哪里呢：
1.内容讲解很浅显，如果说是给新手看的，感觉还不如国人自己写的《21天学通Java》之类的书；
2.练习题也无聊，感觉也不如《21天学通Java》之类的书里面的练习题。难道是因为我不懂老外的文化？
3.好多重要的概念都不讲。变量类型章节起码得说下类型转换的问题吧，类的成员变量修饰符至少得详细说下四种修饰符的作用和区别吧（这里竟然说附录里会详细介绍）。。。

值得赞扬

那么，毕竟是Head First系列的书，难道就没有任何优点吗？我想说还是有一点的：
1.可以看得出来：作者极力用幽默诙谐的话来写一本技术书籍，很多地方的笑话故事还行吧。但是也有一些感觉超级无聊，当然也有中文翻译的问题

我的评分

总体评分：1.0

评分范围0.0-5.0，那么为啥给1.0呢？这本书确实是烂的可以，但是谁又知道有没有更烂的呢，就给以后遇到的更烂的留点位置吧！

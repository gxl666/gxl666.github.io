Dos

初学dos

说是Dos，其实就是win7下的命令行，这个应该不叫Dos吧！但是叫它 cmd 又感觉很奇怪，还是暂时叫Dos吧。

这次之所以要学习Dos，是因为想自己写一个脚本从某些图片网站拉取图片作为自己的电脑桌面壁纸，不至于整天对着一张图片审美疲劳。虽然电脑上也有 360桌面 这种可以定时更换桌面壁纸的软件，但是感觉里面的图片好低俗、好无聊，开机一直运行着还嫌它占内存。所以想自己用脚本写一个，更换完壁纸就自动退出。
首先，找到了 cmd 中自带的帮助文档：

bat.txt
这个帮助文档太简洁了，然后又着了篇教程：教程。虽然作者从别处复制粘贴的很烂，但是看起来还是浅显易懂的，看了 5 个小时，总算大致看完了。后来发现 cmd 下也可以看到每条命令的详细用法：

不过 cmd 貌似只有 ftp 下载功能，而自己又不会用，于是去下载了一个 wget 工具。
然后看了下网上的教程，试了很多，win7 64位 家庭普通版 都不能实现批处理脚本中更换桌面壁纸。以后有时间了再折腾吧。
